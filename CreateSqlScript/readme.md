**설명**
 - DB의 테이블정보를 읽어서 INSERT/UPDATE/MERGE구문으로 변경을 해주는 프로그램이다. 데이터 마이그레이션 및
 - 특정 테이블 백업용으로 사용할 수 있다.

**설정값**
 - database: database 연결정보 
 - settings: 설정값 정보 
 - parameters: 파라미터 정보 
 - query: 실행할 쿼리 정보


**실행방법**
1. db.yml 파일을 속성값에 맞는 정보를 입력한다.
2. run.bat 파일을 실행한다.
3. 결과값이 output 폴더에 저장된다.
4. 저장된 sql을 실행할 DB에서 실행한다.
